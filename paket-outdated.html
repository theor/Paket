Paket

paket outdated

Lists all dependencies that have newer versions available.

1: 

paket outdated [--ignore-constraints] [--include-prereleases|--pre]

Options:

--ignore-constraints: Ignores the version requirement as in the paket.dependencies file.

--include-prereleases [--pre]: Includes prereleases.

If you add the -v flag, then Paket will run in verbose mode and show detailed information.

With --log-file [FileName] you can trace the logged information into a file.

Sample

Consider the following paket.dependencies file:

2: 
3: 
4: 

source https://nuget.org/api/v2

      Castle.Core (>= 2.0.0)
